Image Classification Text Generation Translation Sentiment Analysis Question Answering
Image Classification
POST /api/image_classify

Description: Classify images as Male or Female using CNN

Parameters:

  • image (file, required) - Image file to classify

Response:

{
  "label": "male",
  "score": 0.95,
  "raw_predictions": [0.05, 0.95]
}
Text Generation
POST /api/text_generate

Description: Generate text using GPT-2 transformer model

Parameters (JSON):

  • prompt (string, required) - Starting text for generation

Response:

{
  "text": "One day Pakistan will be a nation of..."
}
Translation
POST /api/translate

Description: Translate English to Urdu using Helsinki-NLP transformer

Parameters (JSON):

  • text (string, required) - English text to translate

Response:

{
  "text": "پاکستان ایک خوبصورت ملک ہے"
}
Sentiment Analysis
POST /api/sentiment_voice

Description: Convert voice to text and analyze sentiment

Parameters:

  • audio (file, required) - Audio file (MP3, WAV, etc.)

Response:

{
  "text": "I love this amazing application!",
  "sentiment": {
    "label": "POSITIVE",
    "score": 0.98
  }
}
Question Answering
POST /api/qa_voice

Description: Convert question from voice to text, answer it, and convert answer back to voice

Parameters:

  • audio (file, required) - Audio file with question

Response:

Audio file (MP3 format) containing the answer

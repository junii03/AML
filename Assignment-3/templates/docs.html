Image Classification
POST /api/image_classify

predicts gender (Male/Female) from an uploaded image using a CNN model.

Parameters (Multipart/Form-Data)
  • image (file, required) - The image file to analyze.
Response
{
  "label": "male",
  "score": 0.98,
  "raw_predictions": [0.02, 0.98]
}
Text Generation
POST /api/text_generate

Generates text continuation based on a prompt using GPT-2.

Parameters (JSON)
  • prompt (string, required) - Initial text to start generation.
Response
{
  "text": "The future of AI involves..."
}
Translation
POST /api/translate

Translates English text to Urdu.

Parameters (JSON)
  • text (string, required) - English text to translate.
Response
{
  "text": "یہ ایک مثال ہے"
}
Sentiment Analysis
POST /api/sentiment_voice

Transcribes audio and performs sentiment analysis on the text.

Parameters (Multipart/Form-Data)
  • audio (file, required) - Audio file containing speech.
Response
{
  "text": "I am feeling great today",
  "sentiment": {
    "label": "POSITIVE",
    "score": 0.99
  }
}
Question Answering
POST /api/qa_voice

Answers a question asked in an audio file, using provided or default context.

Parameters (Multipart/Form-Data)
  • audio (file, required) - Audio file containing the question.
  • context (string, optional) - Context paragraph to answer from. Defaults to info about Pakistan if omitted.
Response

Returns an audio file (MP3) containing the spoken answer.

API Reference

Explore and integrate our available ML endpoints

/api/image_classify

Predicts gender (Male/Female) from an uploaded image using a CNN model.

Parameters (FormData)
  • image (file) - Required image file
Sample Response
{
  "label": "male",
  "score": 0.98
}

/api/text_generate

Generates text continuation based on a prompt using GPT-2.

Parameters (JSON)
  • prompt (string) - Text to start generation
Sample Response
{
  "text": "The future of AI..."
}

/api/translate

Translates English text to Urdu.

Parameters (JSON)
  • text (string) - English text to translate
Sample Response
{
  "text": "یہ ایک مثال ہے"
}

/api/sentiment_voice

Transcribes audio and performs sentiment analysis on the text.

Parameters (FormData)
  • audio (file) - Audio recorded in browser
Sample Response
{
  "text": "Great day!",
  "sentiment": {
    "label": "POSITIVE",
    "score": 0.99
  }
}

/api/qa_voice

Answers a question asked in audio, using provided or default context.

Parameters (FormData)
  • audio (file) - Recorded question
  • context (string, optional) - Custom text
Response Type
Returns audio/mpeg (MP3)
